The Permissions-Policy header was previously named Feature-Policy. As of June 2021, the {{domxref('FeaturePolicy')}} API as well as some linked-to articles and demos still use the older name.

The HTTP {{HTTPHeader("Permissions-Policy")}} header gyroscope directive controls whether the current document is allowed to gather information about the orientation of the deviceÂ through the {{domxref("Gyroscope")}} interface.

Syntax

Permissions-Policy: gyroscope <allowlist>;
<allowlist>
A list of origins for which the feature is allowed. See Permissions-Policy.

Default policy

Default allow list for gyroscope is 'self'.
