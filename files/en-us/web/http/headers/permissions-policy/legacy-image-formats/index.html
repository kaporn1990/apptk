The Permissions-Policy header was previously named Feature-Policy. As of June 2021, the {{domxref('FeaturePolicy')}} API as well as some linked-to articles and demos still use the older name.

The HTTP {{HTTPHeader("Permissions-Policy")}} header legacy-image-formats directive controls whether the current document is allowed to display images in legacy formats.

Syntax

Permissions-Policy: legacy-image-formats <allowlist>;
<allowlist>
A list of origins for which the feature is allowed. See Permissions-Policy.

Default policy

Default allow list for legacy-image-formats is 'self'.
