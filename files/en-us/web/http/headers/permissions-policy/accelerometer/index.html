The Permissions-Policy header was previously named Feature-Policy. As of June 2021, the {{domxref('FeaturePolicy')}} API as well as some linked-to articles and demos still use the older name.

The HTTP {{HTTPHeader('Permissions-Policy')}} header accelerometer directive controls whether the current document is allowed to gather information about the acceleration of the deviceÂ through the {{domxref('Accelerometer')}} interface.

Syntax

Permissions-Policy: accelerometer <allowlist>;
<allowlist>
A list of origins for which the feature is allowed. See Permissions-Policy.

Default policy

The default allowlist value for this feature is: 'self'.
