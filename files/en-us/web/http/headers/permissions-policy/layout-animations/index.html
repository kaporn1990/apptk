The Permissions-Policy header was previously named Feature-Policy. As of June 2021, the {{domxref('FeaturePolicy')}} API as well as some linked-to articles and demos still use the older name.

The HTTP {{HTTPHeader("Permissions-Policy")}} header layout-animations directive controls whether the current document is allowed to show layout animations.

Syntax

Permissions-Policy: layout-animations <allowlist>;
<allowlist>
A list of origins for which the feature is allowed. See Permissions-Policy.

Default policy

Default allow list for layout-animations is 'self'.
